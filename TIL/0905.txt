이벤트
    웹페이지에서 마우스를 클릭했을 때, 키를 입력했을 때, 특정요소에 포커스가 이동되었을 때 어떤 사건을 발생시키는 것

이벤트 종류
    마우스 이벤트
        - click
        - mouseover
        ...

    키 이벤트
        - keydown
        - keyup     : 키를 떼었을 때
        - keypress  : 키를 누른 상태에서
    
    폼 이벤트
        - focus     : 요소에서 포커스가 이동되었을 때
        - blur      : 요소에서 포커스가 벗어났을 때
        - change    : 요소에 값이 변경 되었을 때
        - submit
        - reset
        - select

    로드 및 기타 이벤트
        - load      : 페이지의 로딩이 완료되었을 때
        - abort     : 이미지의 로딩이 중단되었을 때
        - unload    : 페이지가 다른 곳으로 이동될 때
        ...

이벤트 연결
    - 사용자가 실제 이벤트를 발생시켰을 때 그 이벤트에 대응하여 처리하는 것을 이벤트 핸들러
    - 앞에 on을 붙여주고 이벤트에 대한 동작(함수)을 처리


addEventListener()
    document의 특정요소(Id, class, tag, ...) event를 등록할 때 사용한다.
