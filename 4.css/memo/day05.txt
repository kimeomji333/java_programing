float
	HTML 요소가 주변(수평으로 나열된)의 다른 요소들과 자연스럽게 어울리도록 만들어줌
	float를 적용받은 요소의 다음 요소들은 끌어 올려짐
	float를 적용받은 요소의 방향을 결정(left, right)
	float를 적용받은 요소는 다른 요소보다 위쪽에 위치
	박스들을 수평방향으로 정렬할 시 사용
	float를 적용받은 요소는 컨텐츠 크기 만큼만 영역을 설정
	
clear	
	float 속성이 적용된 이후 나타나는 요소들의 동작을 조절
	float 속성이 적용되면 그 이후에 등장하는 요소들의 정확한 위치를 설정하기 어려움
	clear 속성을 사용하여 이후에 등장하는 요소들이 더 이상 float 속성에 영향을 받지 않도록 설정
	어느 방향의 float에 영향을 받지 않게 할 것인지에 따라 left, right, both 설정

header "내부 요소"들 배치시에 position 사용해도 됨
position 공부해두자. 부모 안에서 어디에 위치해야하는 지 알아야하기 때문에 위치를 계산해야함
하나씩 position으로 배치시키면 계산해야 하니, display: flex를 이용해 내부에 있는 요소들을 가로로 배치시킨다.

플렉서블 레이아웃(flex)
	대부분의 웹 사이트는 수직 구성
	레이아웃을 구성할 때 가장 많이 이용하는 요소들은 대부분 블록 요소
	수직 구성은 쉽게 구성할 수 있으나, 수평 구성은 구현하기 쉽지 않음
	과거에는 수평 구성을 <table>, float, inline-block 으로 구성
	좀 더 편하고 쉽게 수평 구성을 구현하기 위해 flex를 사용한다. 

flex-wrap
	플렉스 라인에 더 이상 여유가 없을 때 플렉스 요소들의 위치를 어떻게 할지 설정

	nowrap: 기본 설정, 플레스 요소가 다음줄로 넘어가지 않음, 요소의 너비를 줄여 한 줄에 모두 배치
	wrap: 플렉스 요소가 여유 공간이 없으면 다음 줄로 넘어감
	wrap-reverse: 플렉스 요소가 여유 공간이 없으면 이전 줄로 넘어감(위쪽으로 넘어감)

justify-content
	플렉스 요소의 수평방향 정렬 방식을 설정

	flex-start	: 기본 설정, 앞쪽에서부터 배치됨
	flex-end	: 뒤쪽에서부터 배치됨
	center		: 가운데에 배치됨
	space-between	: 요소들 사이에 여유 공간을 두고 배치됨
	space-around	: 앞, 뒤, 그리고 요소들 사이에 모두 여유 공간을 두고 배치됨
	space-evenly	: 앞, 뒤, 그리고 요소들 사이에 공간을 모두 일정하게 두고 배치됨

align-items
	플렉스 요소의 수직방향 정렬 방식을 설정
	
	stretch		: 기본설정, 플렉스 요소의 높이가 컨테이너의 높이와 같게 변경된 뒤 연이어서 배치됨
	flex-start	: 플렉스 요소는 플렉스 컨테이너의 위쪽에 배치됨(기준선은 위쪽 선)
	flex-end	: 플렉스 요소는 플렉스 컨테이너의 아래쪽에 배치됨(기준선은 아래쪽 선)
	center		: 플렉스 요소는 플렉스 컨테이너의 가운데에 배치됨
	baseline	: 플렉스 요소는 플렉스 컨테이너의 기준선에 배치됨(요소들의 content 기분으로 기준선을 잡음)

align-self
	플렉스 요소마다 서로 다른 align 속성값을 설정할 수 있다.(수직 방향 정렬)

order
	플랙스 요소들의 순서를 설정

flex-grow
	플렉스 아이템들이 차지할 너비에 대한 증가형 숫자를 설정

flex-shrink
	플렉스 아이템들이 차지할 너비에 대한 감소형 숫자를 설정

flex-basis
	플렉스 아이템들의 기본 길이를 지정
flex
	flex 의 속성을 한꺼번에 설정
	flex: 1 1 0;	(← 쓸일은 거의 없다. 보통 이렇게 쓰이고 있다고만 알아두기)
	grow shrink basis
	컨테이너에 설정, 컨테이너 안의 요소들은 수평 정렬이 되고(내부의 요소에 영향을 받으니, align-self부터 사용함.)
