함수
	객체가 가진 것은 메소드, 그렇지 않으면 함수
	하나의 특별한 목적의 작업을 수행하도록 설계된 독립적인 영역을 의미한다.
	자바스크립트는 함수도 하나의 타입이다.
	따라서 함수를 변수에 대입할 수 있다. 

	function 함수명(매개변수1, 매개변수2, ...) {
		실행할 문장
		return 리턴값
	}

디폴트 매개변수
	매개변수에 매개값을 넘겨주지 않으면 대신해서 기본값을 매개변수에 대입 후
	함수 내부의 문장을 수행하게 할 수 있다.

	function 함수명(매개변수1 = 값, 매개변수2 = 값, ...){
		...
	}

나머지 매개변수
	생략 접두사(...)를 사용해서 특정 위치의 매개변수부터 마지막 매개변수까지 
	한 번에 "배열"로 받아올 수 있다.

	function 함수명(...매개변수){
		...
	}

함수 표현식
	const 함수명 = function(매개변수1, 매개변수2, ...){

	}
	자바스크립트는 함수를 변수에 저장할 수 있으므로 익명함수를 선언하여
	특정 변수에 대입해준다.

호이스팅
	변수 선언 단계 : 3가지 선언, 할당,  = 공간 확보, 집 짓기, 들어가기
	
	예시)
	data = 10;
	var data; // var는 이 선언문을 위로 끌어올려서 작성하기 때문에
	
	자바스크립트의 함수 선언이나 var 변수의 경우 선언시 호이스팅이 일어난다.
	선언문들을 가장 최상단으로 끌어올려서 실행하는 것처럼 작동하는 것을 호이스팅이라 한다.
	
	함수 안에 필요한 변수들을 모두 모아서 유효범위(scope)의 최상단에 선언을 한다.
	
화살표 함수(람다식)
	function 키워드를 사용하여 함수를 만드는 것보다
	간단하게 함수를 선언할 수 있다.
	람다식은 항상 익명함수이다.
	객체의 메소드로는 사용하지 않는다.

	const 함수명 = (매개변수1, 매개변수2, ..) => {
		...
	}
