연산자
	10 == '10'	-> true
	10 === '10'	-> false

	=== : 같다(타입과 값이 모두 같아야 함)
	!== : 다르다(타입, 값 둘 중 하나라도 다르면 참)

배열(Array)
	자바스크립트의 배열은 Array 타입의 객체

배열의 선언
	const 배열명 = [값1, 값2, ...];
	const 배열명 = new Array(값1, 값2, ...);

Array 객체의 메소드
	push()		배열의 마지막에 요소를 추가
	pop()		배열의 마지막에 있는 요소을 제거(제거된 값을 리턴)
	shift()		배열의 첫번째에 있는 요소을 제거(제거된 값을 리턴)
	concat()	두 개의 배열을 연결하여 새로운 배열 생성(생성된 배열을 리턴)
	join()		배열 요소 사이마다 원하는 문자를 삽입해서 문자열 생성(생성된 문자열을 리턴)
	reverse()	배열을 역순으로 재배치
	sort()		배열을 정렬

		}

	for of 문
		변수가 뒤에 오는 반복자의 값들을 꺼내오며 반복
	
		for(const 변수 of 반복자객체){

		}
