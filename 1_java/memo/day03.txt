배열
	같은 자료형의 여러 변수들을 나열해서 선언해 놓은 것


배열의 선언
	자료형[] 배열명 = {값1, 값2, ... };
	자료형[] 배열명 = new 자료형[칸수];

배열의 사용
	data = 10;
	syso(data);
	
	배열명[방번호] = 값;
	syso(배열명[방번호]);

배열의 사용 이유
	1. 비슷한 의미의 값들을 하나의 이름과 방번호(인텍스)로 관리하기 위해서
	2. 규칙성이 없는 여러 값들에 규칙성을 부여하기 위해서

배열의 길이
	배열명.length
	

배열의 인덱스 번호
	실제 값들이 저장될 수 있는 공간이 할당되면 이름은 존재하지 않고 "주소만 존재"한다. 
	따라서 각 공간들에 쉽게 접근할 수 있도록 첫 번째 시작 주소를 배열명 변수가 기억하고 있고,
	주소연산(추가 이동)?을 통해 다음 주소로 이동한다. 
	시작 주소에서 연산된 정수(이동한 횟수)가 해당 방의 인덱스 번호가 되고
	첫 번재 방은 배열명 변수가 가지고 있는 시작 주소에서 0번 추가 이동 해야 하므로
	인덱스 번호는 0번 부터 시작한다. 


2차원 배열
	배열 안에 배열이 있는 형태
	자료형[][] 배열명 = {
		{값1, 값2, ... },
		{값3, 값4, ... },
		...
	}

	자료형[][] 배열명 = new 자료형[행수][열수];

	정방배열	:각 행마다 열의 개수가 모두 같은 배열
	비정방배열	:각 행마다 열의 개수가 다른 배열
