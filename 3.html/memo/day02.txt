웹 접근성
	정상적인 웹 컨텐츠 사용이 가능한 일반 사용자부터 고령자, 장애인 같은 신체적, 환경적 조건에 제한이 있는 사용자들을
	포함해 모든 사용자들이 동일한 조건에서 접근할 수 있도록 
	웹 컨텐츠를 제작하는 기법

웹 표준
	W3C에서 제정
	여러 브라우저에서 동일한 사용자 경험을 줄 수 있도록 하는 기술, 방법 등을 의미/크로스 브라우징
	
서식 태그
	<서식태그>내용</서식태그>
	
	<strong>
		텍스트를 굵게 표현하고 싶을 때 사용
		<b>태그와 동일한 기능
		웹 접근성에 사용
	<em>
		텍스트를 이탤릭체로 표현하고 싶을 때 사용
		<i>태그와 동일한 기능
		웹 접근성에 사용
	<mark>
		텍스트에 하이라이팅(형광)효과를 적용
	<del>
		텍스트를 중앙에 가로줄을 만들어 텍스트를 지운 것과 같은 효과를 줌
	<ins>
		텍스트 밑에 선을 그어 텍스트에 강조효과를 줌
	<sup>, <sub>
		위첨자, 아래첨자

HTML 요소의 속성
	<태그명 속성명="속성값" 속성명="속성값" ... >내용</태그명>
	속성은 해당 요소에 대한 추가적인 정보를 제공
	속성에 대한 값은 따옴표로 꼭 묶어주지 않아도 되지만, 속성값에 띄어쓰기가 있는 경우 문제가 발생할 수 있으므로
	따옴표를 쓰기 권장함(쌍따옴표, 홑따옴표 둘 다 가능)

HTML5 의 문서 구조
	<!DOCTYPE html>
		HTML 문서의 선언 / HTML 버전 5를 나타냄
	<html lang="en">
		lang 속성은 웹 접근성에 관한 명시
		스크린 리더기에서 언어를 인식하여 자동으로 음성을 변환하거나, 
		해당 언어에 적합한 발음을 제공(한국어 설정은 lang="ko")

	<meta> 태그
		<head> 태그에 정보를 추가하기 위한 태그
		
		charset="UTF-8" 
			charset 속성은 문자를 인코딩할 때 설정하는 역할
			우리나라는 윈도우 기본 문자셋은 EUC-KR을 사용해도 되지만,
			다국어로 인코딩하기 위해서는 UTF-8 문자셋 사용하기를 권장
			EUC-KR은 완성형, UTF-8은 조합형

		name="viewport" content="width=device-width, initial-scale=1.0"
			viewprot: 가상의 화면 
			모바일로 최적화된 사이트에 포함하는 META 속성 설정
			key-values 구조
			{viewport: content, initail-scale}
			
비트맵과 벡터 이미지
	비트맵 이미지(like 점묘화)
		픽셀이 모여서 만들어진 정보의 집합
		레스터 이미지라고 부름
		픽셀 단위로 화면에 렌더링함
		그림판, 포토샵 등 툴로 편집

	벡터 이미지
		수학적 정보의 형태들이 만들어내는 결과물
		이미지가 가지고 있는 점, 선, 면의 위치, 색상 정보, 크기 등의 가지고 있음
		확대 및 축소를 해도 이미지가 깨지지 않음
		일러스트 같은 툴로 편집

이미지 파일의 형식
	jpg(jpeg)
		압축률이 뛰어나 사진이나 예술분야에 많이 사용
		가장 널리 쓰이는 이미지 포맷
		표현 생상(24비트, 약 1600만 색상)이 뛰어나 고해상도 표시장치에 적합
		핸드폰, 카메라 사진
		손실 압축

	gif
		파일 내에 다른 이미지 및 문자열 같은 정보를 저장할 수 있는 파일
		여러장의 이미지를 한 개의 파일에 저장할 수 있음(움짤, 애니메이션)
		8비트 컬러(256 색상)만 지원
		비손실 압축

	png
		gif의 대체 포맷으로 개발
		8비트, 24비트 컬러 이미지 처리
		w3c 권장 포맷
		알파 채널 지원(추가적인 채널 지원한다는 뜻) - 배경을 투명하게 설정할 수 있다.
		아이콘

	webp
		jpg, png, gif를 모두 대체할 수 있는 구글이 개발한 이미지 포맷
		완벽한 포맷
		gif같은 애니메이션 지원
		알파 채널 지원
		손실, 비손실 압축

table
	여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표
	행과 열로 이루어져 있다. 
	
	<table>
		<tr>			<- t row 테이블의 각 층(행)
			<td>내용</td>	<- t data 테이블의 각 셀
			<td>내용</td>
			...
		</tr>
		...
	</table>

<table>의 속성
	border	: 테이블의 테두리 두께를 설정(기본 0)
	width	: 테이블 전체의 가로 폭을 설정
	align	: 테이블의 정렬을 설정(left, right, center)

<tr>, <th>, <td>의 속성
	width	: 셀의 가로 폭을 설정(td/th에 주면: 같은 열 전부 적용)
	height	: 셀의 세로 폭을 설정(td/th에 주면: 같은 행 전부 적용)
	align	: 셀의 텍스트 정렬을 설정(td/th에 주면: 해당 셀만 적용)
